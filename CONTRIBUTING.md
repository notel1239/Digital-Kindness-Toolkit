# Contributing to Digital Kindness Toolkit

Thank you for your interest in contributing to this community-driven resource hub! We welcome contributions from everyone, regardless of experience level. Here's how you can help.

## Ways to Contribute

### 1. **Suggest New Resources**
   - Share links to Philippine-based hotlines, legal aid, mental health support, online safety guides, etc.
   - Provide descriptions and why the resource is valuable.

### 2. **Share Personal Stories (Anonymously Optional)**
   - We believe in the power of lived experiences to inspire hope and solidarity.
   - You can share your story of overcoming online harassment, cyberbullying, or digital rights challenges.
   - Anonymity is respectedâ€”please indicate if you wish to remain anonymous.

### 3. **Propose Campaign Ideas**
   - Suggest ideas for social media campaigns, awareness drives, or educational materials.
   - Outline the goal, target audience, and possible execution steps.

### 4. **Improve Existing Content**
   - Fix typos, improve clarity, or expand sections in the documentation.
   - Add translations (Filipino/Tagalog, regional languages) to make resources more accessible.

### 5. **Review Pull Requests**
   - Help review contributions from others to ensure quality and accuracy.

## How to Contribute (Technical Steps)

1. **Fork the Repository**
   - Click the "Fork" button at the top right of this repository page.

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/your-username/Digital-Kindness-Toolkit.git
   cd Digital-Kindness-Toolkit
   ```

3. **Create a Branch**
   ```bash
   git checkout -b your-branch-name
   ```
   Use descriptive branch names (e.g., `add-mental-health-hotlines`).

4. **Make Your Changes**
   - Edit existing files or create new ones.
   - Follow the existing style and formatting.

5. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Brief description of your changes"
   ```

6. **Push to Your Fork**
   ```bash
   git push origin your-branch-name
   ```

7. **Open a Pull Request (PR)**
   - Go to the original repository on GitHub.
   - Click "Pull Requests" â†’ "New Pull Request".
   - Select your branch and provide a clear title and description.
   - Link any related issues by writing "Closes #issue-number".

## Guidelines for Content

- **Respect Privacy**: Do not share personally identifiable information without explicit consent.
- **Be Kind and Inclusive**: Use language that is respectful, empathetic, and free from discrimination.
- **Fact-Check**: Ensure resources are accurate and up-to-date.
- **Focus on Empowerment**: Aim to provide actionable, supportive, and hopeful content.

## Code of Conduct
Please read our [Code of Conduct](code_of_conduct.md) before participating. We are committed to providing a welcoming and harassment-free environment for everyone.
